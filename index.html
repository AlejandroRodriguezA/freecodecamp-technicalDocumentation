Formas para vincular CCS al HTML

CCS (Cascading Style Sheets) le dice al navegador como mostrar el texto y las etiquetas de HTML. Con CSS puedes controlar las propiedades y aspectos como el color, ubicación, tamaño, entre otros.

  • Usando documento externo:
    • Usando la etiqueta <link rel="stylesheet" href="style.css">.
    • Usando la regla @import.
  • Dentro del HTML:
    • Usando la etiqueta o tag (en ingles): <style> ... </style>.
    • Estilo en-linea (Inline Style).
  • Usando JavaScript.
Usando documento externo

Un documento CSS externo puede ser aplicado a varios HTML y puede ser implementado de dos formas:

  • Usando la etiqueta <tag></tag>. Como por ejemplo:

    • Hellow-word.html

    
    <!DOCTYPE html>
    <html> 
        <head>
            <meta charset="utf-8" /> 
            <link rel="stylesheet" type="text/css" href="style.css"> 
        </head>
        <body>
            <h1>Hello world!</h1> 
            <p>I ♥ CSS</p> 
        </body> 
    </html>

    style.css

    
    h1 {
        color: green; 
        text-decoration: underline; 
    } 
    p {
        font-size: 25px;
        font-family: 'Trebuchet MS', sans-serif; 
    }
  • Usando la regla @import. Esta etiqueta tiene que preceder todas las otras reglas, excepto @charset, y @import no puede ser usado en condicionales. Esto, usualmente, es usado para importar estilos o tipos o fuentes de letras, como por ejemplo:
    • @import 'https://fonts.googleapis.com/css?family=Lato';

    Otro ejemplo:

    
    <style> 
        @import url('/css/styles.css');
    </style>
Dentro del HTML

  • Usando la etiqueta (tag) <style></style> dentro del HTML funciona igual que en los documentos externos, con la única diferencia que al estar dentro de HTML no puede ser usado por otro HTML.

    Suele estar en la sección <head>, pero puede estar en cualquier parte del HTML, pero lo recomendado es en la cabecera, como por ejemplo:

    
    <head>
        <style>
            h1 {
            color: green;
            text-decoration: underline;
            }

            p{
                font-size: 25px;
                font-family: 'Trebuchet MS', sans-serif; 
            }
        </style>
    </head>
    <body>
        <h1>Hello world!</h1> 
        <p>I ♥ CSS</p> 
    </body>

  • Los estilo en-linea (Inline Style) se usan o aplican directamente en el elemento, pero esto no es una practica recomendada, pues cualquier cambio que se necesite hacer tienes que recorrer todo el HTML para ubicar y cambiar.

    Aquí un ejemplo:

    
    <h1 style="     color: green; 
                    text-decoration: underline;">
        Hello world!
    </h1> 

    <p style="font-size: 25px; 
                font-family: 'Trebuchet MS';">
        I ♥ CSS
    </p>
Usando JavaScript

Es posible agregar, borrar o cambiar las propiedades de CSS con JS, como por ejemplo:


    var el = document.getElementById("element"); 
    el.style.opacity = 0.5;
    el.style.fontFamily = 'sans-serif';

Nota que la propiedad es nombrada usando “lower camel case” (https://es.wikipedia.org/wiki/Camel_case). En el ejemplo se puede ver la propiedad “font-famify” , en JS se usa fontFamily.

Adicionalmente, se pueden utilizar librerías externas con JQUERY, entre otras.
